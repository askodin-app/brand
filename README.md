## Usage Guidelines

These assets are provided for authorized use in connection with [askOdin](https://askodin.app) products, partnerships, and press coverage.

**Do:**
- Use the provided SVG or high-resolution PNG files
- Maintain the original color combinations
- Ensure adequate clear space around the logo (minimum 1x the height of the "O" in Odin)
- Use the colored variant on both light and dark backgrounds

**Don't:**
- Alter the logo colors, proportions, or spacing
- Add effects (shadows, gradients, outlines) to the logo
- Place the logo on busy or low-contrast backgrounds
- Recreate or redraw the logo from scratch
- Use the logo to imply endorsement without permission

For press inquiries, partnership branding, or custom asset requests, contact [hi@askodin.app](mailto:hi@askodin.app).
